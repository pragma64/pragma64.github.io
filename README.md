# pragma64.github.io

GitHub Pages site for **pragma64.com**.

## Structure

```
CNAME                          # Custom domain config
index.html                     # Project index / landing page
projects/
  clojure-prototype/           # ClojureScript 3D prototype (Three.js)
  void-wanderer/               # Void Wanderer (single-file bundle)
```

Each subdirectory under `projects/` is a self-contained app served via its own `index.html`.
